# Superlean Silex Starterplate

- web
- api
- console

## Install
- composer create-project
- bower install

## Requirements
- bower

## Features
- console provider
- http_cache
- dotenv
- twig

### Debug
- symfony vardumper and stop
- symfony errors

## Run Web

    php -S localhost:8080 -t web web/index.php

## Run Api

    php -S localhost:8088 -t web web/api.php
    
## Run Cli

    php console hello-world
